🎓 Student Grade Management System (Python)

A simple command-line based Student Grade Management System built using Python. This project allows users to manage student records, calculate grades, and display performance summaries efficiently.

📌 Features
Add new student records
View all students
Update student marks
Delete student records
Calculate grades automatically
Display individual and overall performance
🛠️ Technologies Used
Python 3
Standard Python libraries (no external dependencies)
